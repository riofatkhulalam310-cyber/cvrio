# Rio Fatkhul Alam Hamizan — Personal Portfolio Website

Website CV/portofolio interaktif dengan background 3D (Three.js), menampilkan profil sebagai Web Developer, Cybersecurity enthusiast, dan Trader berbasis Smart Money Concept (SMC).

---

## 📂 Struktur Project

```
web-cv/
├── index.html       Halaman utama website (rename dari cv-rio.html untuk GitHub Pages)
├── cv-rio.html       File asli (bisa dihapus setelah di-rename)
├── fotorio.jpg       Foto profil
└── README.md         Dokumentasi ini
```

> **Catatan penting:** Untuk deploy ke GitHub Pages, file `cv-rio.html` harus di-rename menjadi `index.html` agar otomatis terbaca sebagai halaman utama.

## ✨ Fitur

- **3D Interactive Background** — animasi constellation/node network dengan Three.js, bereaksi terhadap scroll dan gerakan mouse

- **Dark Theme Modern** — palet warna navy gelap dengan aksen biru & teal
- **Single File Deployment** — seluruh halaman dalam satu file HTML, tidak perlu proses build

---

## 🌐 Deployment ke GitHub Pages

1. Rename `cv-rio.html` menjadi `index.html`
2. Upload seluruh isi folder (`index.html`, `fotorio.jpg`, `README.md`) ke repository GitHub
3. Masuk ke **Settings → Pages**
4. Pilih branch `main`, folder `/ (root)`, lalu **Save**
5. Website live di:
   ```
   https://riofatkhulalam310-cyber.github.io/nama-repo/
   ```
